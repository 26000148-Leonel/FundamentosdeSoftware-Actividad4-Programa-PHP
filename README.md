# Mostrar Nombre en PHP e imagen creada con IA
##  En este respositorio estaremos adjuntando el codigo sobre un programa el cual nos permitira ver en el navegador a traves de un aruchivo HTML, CSS y PHP, imprime en la pagina :Hola Mundo soy Dylan Leonel Andrade Lucas" y se muestra una imagen creada con IA y agregada a travez de la etiqueta img y estilizadas a traves de CSS. Todo esto fue realizado a traves de la maquina virtual.
### Las tecnologias a utilizar para realizar este programa serian instalar PHP, deberemos intalar visual studio code en caso no lo tengamos instalado, ademas deberemos instalar Git y crear una cuenta en GitHub o utilizar una cuenta ya existente, ademas del Apache HTTP Server.
### Pasos para su ejecucion
### 1) Abrir Studio code
### 2) Abrir la ruta /var/www/html/
### 2) Otorgarle permisos a traves de la terminal
### 3) Crear un archipo PHP
### 4) Realizar el codigo que se solicita
### 5) Abrir en el navegador la ruta
### 6) Ruta del archivo: 
http://localhost/carpeta_php/index.php
### 7) Crear un Repositorio en GitHub
### 8) Ir a la carpeta del proyecto Ejemplo: cd /var/www/html/carpeta_php
### 9) Inicializar el repositorio Git Ejemplo: git init
### 10) Agregar los archivos al repositorio Ejemplo: git add .
### 11) Crear el commit Ejemplo: git commit -m "First Commit"
### 12) Conectar el repositorio con GitHub Ejemplo: git remote add origin https://github.com/USUARIO/NOMBRE_REPOSITORIO.git
### 13) Subir los archivos a GitHub Ejemplo: git push -u origin main
### 14) Abrir el repositorio, validar que este cargados todos los archivos.
